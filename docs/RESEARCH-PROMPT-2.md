אני בונה תחשיב עלות-שחזור (replacement cost) לפרויקט תוכנה, ואני צריך נתוני שוק עדכניים
(2024–2025) להשלמת התעריפים של מספר תפקידים שלא כוסו בדוח קודם. אנא ספק טבלאות לפי אותם
שווקים: ארה"ב (In-house), ישראל (In-house), דובאי/איחוד האמירויות, אוקראינה (Outstaffing/FOP),
ופקיסטן (Agency/Outsourcing).

לכל תפקיד ובכל שוק ספק: **שכר חודשי ברוטו**, **עלות חודשית מלאה למעסיק (כולל תקורה)**,
ו**תעריף שעתי מלא**. צטט מקורות.

## חלק 1 — תפקידי הנדסה/מוצר חסרים
1. **Security / Cryptography Engineer** — מהנדס אבטחה והצפנה (AES, חתימות דיגיטליות, JWT/HMAC,
   web3/WalletConnect). מומחיות נדירה; ציין אם יש פרמיית שכר מעל Backend רגיל ובכמה אחוזים.
2. **Engineering Manager / Tech Lead** — מוביל טכני / מנהל פיתוח (ארכיטקטורה, code review, ניהול צוות).
3. **Frontend / Web Engineer** — מפתח web (אתרי שיווק + פורטלים), בנפרד מ-Mobile React Native.
   ציין אם התעריף שונה מ-Mobile RN.
4. **Growth / Marketing Manager** — שיווק/צמיחה (קופי, נחיתה, תוכן), לא הנדסי.

## חלק 2 — לוקליזציה (פירוט מיוחד — היקף גדול)
ההקשר: אפליקציית מובייל + web, מקור באנגלית, **~1,487 מחרוזות UI (~9,000–12,000 מילים)**.
**לוקליזציה מלאה ל-~33 שפות** (נבנתה בפועל): ערבית, אורדו, פרסית, פשטו, דארי (prs), כורדית (ckb),
עברית — כולן RTL — וכן הינדי, בנגלית, גוג'ראטי, פנג'אבי, טמילית, מלאיאלאם, נפאלית, סינהלה, אמהרית,
האוסה, סינית, יפנית, קוריאנית, תאית, וייטנאמית, אינדונזית, טאגלוג, אוקראינית, רוסית, גרמנית,
צרפתית, ספרדית, איטלקית ועוד.

ספק:
1. **מודלים לתמחור תרגום/לוקליזציה** — פר-מילה, פר-שפה, שעתי, פרויקט. הסטנדרט בתעשייה?
2. **תעריף פר-מילה לתרגום מקצועי** מאנגלית למשפחות השפות הרלוונטיות (ערבית/אורדו/פרסית,
   שפות הודיות, שפות מזרח-אסיה, שפות אירופיות). פרילנסר מול סוכנות.
3. **Software/App localization** מעבר לתרגום גולמי: ניהול מחרוזות, התאמת RTL (ערבית/אורדו/פרסית/
   פשטו/עברית), linguistic QA, התאמות תרבותיות — כמה זה מוסיף לעלות.
4. **עלות מלאה לפרויקט** ללוקליזציה של ~1,500 מחרוזות / ~10k מילים **ל-33 שפות** (טווח),
   וגם עלות פר-שפה כדי שנוכל לכפול.
5. **זמן (timeline)** טיפוסי ללוקליזציה של אפליקציה בהיקף כזה ל-33 שפות — שבועות/חודשים,
   כולל QA לשוני; ציין אם עובדים במקביל על מספר שפות.
6. **תעריף Localization Specialist / Localization PM במשרה** (חודשי + שעתי) ב-5 השווקים.

## חלק 3 — ייצור אססטים ויזואליים (זמן מעצב)
ההקשר: בפרויקט **~498 אססטים ויזואליים** שהופקו: 198 PNG, 156 SVG (אייקונים), 115 WebP,
24 JPG, 5 MP4 — אילוסטרציות, אייקונים מותאמים, תמונות hero, splash, גרפיקת מותג, מקצתן
תמונות שיוצרו ב-AI ולוטשו. שיטת שורות-הקוד לא סופרת זמן זה. ספק **בנצ'מרק זמן + עלות לכל סוג**:
1. **אייקון מותאם (SVG)** — שעות/עלות פר אייקון, וסט אייקונים.
2. **אילוסטרציה מותאמת** — פר אילוסטרציה (פשוטה מול מורכבת).
3. **תמונת hero / banner מלוטשת** (כולל אם נוצרה ב-AI ועברה ליטוש/ריטוש).
4. **Splash / branding / לוגו ומערכת מותג** — פרויקט זהות.
5. **Motion / Lottie / וידאו קצר** — פר נכס מונפש.
6. **תעריף מעצב/אילוסטרטור/מוֹשׁן** (חודשי + שעתי) ב-5 השווקים, אם לא כוסה בדוח הראשון.
