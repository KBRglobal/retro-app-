## חלק 2: לוקליזציה ל-33 שפות

### 2.1 מודלי תמחור בשוק הלוקליזציה (2024–2025)

שוק הלוקליזציה פועל בארבעה מודלי תמחור עיקריים:

| מודל | שכיחות | מתי מתאים |
| :--- | :--- | :--- |
| **לפי מילה (Per-word)** | הנפוץ ביותר | תרגום UI strings, תיעוד עם scope מוגדר |
| **לפי שעה (Hourly)** | נפוץ לשירותי QA | LQA, הסתגלות תרבותית, בדיקות |
| **לפי פרויקט (Project-based)** | נפוץ ב-Enterprise | פתרונות end-to-end כוללים |
| **מנוי / SaaS** | מגמה עולה (Nimdzi 2025) | תוכנות עם עדכוני תוכן שוטפים |

> **מגמה חשובה:** Nimdzi Insights (2025) מדווח על מעבר הדרגתי מתמחור per-word לעבר hourly ו-project-based, עם גידול ב-MTPE (Machine Translation Post-Editing) — כ-46% מתוכניות Enterprise משתמשות ב-MTPE נכון ל-2024.

| **מזרח אסיה** | סינית, יפנית, קורנית, תאית, וייטנאמית | $0.10 – $0.25 | $0.18 – $0.35 |
| **אירופיות** | גרמנית, צרפתית, ספרדית, איטלקית, אוקראינית, רוסית | $0.08 – $0.20 | $0.10 – $0.25 |

- **יפנית וקורנית:** גבוהות יותר בשל מורכבות ומחסור במתרגמים — עד $0.35/מילה בסוכנות.
- **ספרדית, צרפתית, איטלקית:** זולות יחסית בשל היצע גבוה של מתרגמים.
- **MTPE:** 30%–50% זול יותר מתרגום אנושי מלא. מתאים לאפליקציות עם UI strings סטנדרטיים.
